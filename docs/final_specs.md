# Final Specs — AI Scraping Platform

> **Status:** DRAFT — To be completed in Phase 1
> **Last Updated:** 2026-03-22

## 1. Product Vision

This platform is a production-grade, cloud-agnostic AI scraping system that supports:
- Public cloud deployment (SaaS)
- Self-hosted deployment
- Windows EXE (desktop)
- Browser extension
- API integration for third-party apps

The Windows EXE and browser extension are NOT separate products — they are alternate runtime front ends for the same core routing and extraction platform.
